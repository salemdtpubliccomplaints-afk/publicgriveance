Updated files for Public Grievance System.

Copy these files into your project:
- app.py -> project root
- requirements.txt -> project root
- Procfile -> project root
- templates/*.html -> templates folder

Also create these folders if missing:
- static/images
- static/uploads

Keep your existing static/images/logo.png file.

Run this once locally:
python app.py

The database columns for before_photo and after_photo are auto-added by app.py.
